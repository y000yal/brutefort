BruteFort – BruteForce Protection for WordPress with IP Restriction, Whitelist & Blacklist Management. Lock down your login page with smart security.

== Description ==

**BruteFort** is your WordPress site's frontline defense against brute force attacks. This lightweight yet powerful plugin gives you full control over login security by limiting login attempts, blocking suspicious IPs, and letting you manage custom whitelist and blacklist rules.

Whether you're running a blog, a WooCommerce store, or a membership site, BruteFort helps you keep bots, hackers, and bad actors out — without slowing your site down.

= 🔐 Features =
- Block brute force attacks with rate limiting & IP monitoring
- Restrict access to your login page by IP address
- Manage custom **IP whitelists** and **blacklists**
- Instantly block suspicious login attempts
- Set login attempt limits and timeout durations
- Lightweight and performance-friendly
- Works seamlessly with WordPress login system
- Simple, no-nonsense settings panel

= 🎯 Use Case =
Great for site owners, agencies, and anyone who’s tired of getting login spammed or scanned by bots. BruteFort adds an extra layer of security **without relying on heavy firewall plugins**.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/brutefort` directory, or install the plugin through the WordPress plugin screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to **Settings > BruteFort** to configure IP restrictions, whitelist/blacklist, and login attempt limits.

== Frequently Asked Questions ==

= Does this plugin slow down my site? =
Not at all. BruteFort is designed to be lightweight and optimized for performance.

= Can I whitelist my own IP? =
Yes! You can easily add your own IP or a range of IPs to the whitelist to ensure you’re never locked out.

= What happens if someone is on the blacklist? =
Blacklisted IPs are instantly blocked from accessing the login page.

= Is this compatible with other security plugins? =
Yes. BruteFort is designed to work alongside most other WordPress security plugins without conflict.
